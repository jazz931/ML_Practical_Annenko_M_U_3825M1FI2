# Практическая работа №5 по машинному обучению

**Задача**: предсказать покупку (Revenue = True/False) на основе поведения пользователя в интернет-магазине.

**Датасет**: [Online Shoppers Purchasing Intention Dataset](https://www.kaggle.com/datasets/imakash3011/online-shoppers-purchasing-intention-dataset)

**Модели**: kNN, Random Forest, Extra Trees, AdaBoost  
**Метрика качества**: F1-score для класса "Купил"  
**Лучшая модель**: Random Forest (F1 = 0.6640)
